groups of properties are set in a PortfolioCVaR object with various “set” and “add” functions. For
example, to set up an average turnover constraint, use the setTurnover function to specify the
bound on portfolio turnover and the initial portfolio. To get individual properties from a
PortfolioCVaR object, obtain properties directly or use an assortment of “get” functions that obtain
