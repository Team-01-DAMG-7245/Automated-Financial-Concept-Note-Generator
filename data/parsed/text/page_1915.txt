This table lists the default color palettes for plots in the light and dark themes.

Palette
Palette Colors

"gem" — Light theme default

Most plots use these colors by default.

"glow" — Dark theme default

You can get the RGB triplets and hexadecimal color codes for these palettes using the
orderedcolors and rgb2hex functions. For example, get the RGB triplets for the "gem" palette
and convert them to hexadecimal color codes.

RGB = orderedcolors("gem");
H = rgb2hex(RGB);

Get the RGB triplets using RGB = get(groot,"FactoryAxesColorOrder").

Get the hexadecimal color codes using H = compose("#%02X%02X%02X",round(RGB*255)).
