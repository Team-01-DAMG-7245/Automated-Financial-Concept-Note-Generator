cfplot

Visualize cash flows of financial instruments

Syntax

cfplot(CFlowDates,CFlowAmounts)
cfplot(ax,CFlowDates,CFlowAmounts)
cfplot( ___ ,Name,Value)

h = cfplot( ___ ,Name,Value)
[h,axes_handle] = cfplot( ___ ,Name,Value)

Description

cfplot(CFlowDates,CFlowAmounts) plots a cash flow diagram for the specified cash flow
amounts (CFlowAmounts) and dates (CFlowDates). The length and orientation of each arrow
correspond to the cash flow amount.

cfplot(ax,CFlowDates,CFlowAmounts) plots a cash flow diagram for the specified cash flow
amounts using an optional ax argument.

cfplot( ___ ,Name,Value) plots a cash flow diagram for the specified cash flow amounts
(CFlowAmounts), dates (CFlowDates), and optional name-value pair arguments.

h = cfplot( ___ ,Name,Value) returns the handle to the line objects used in the cash flow
diagram.

[h,axes_handle] = cfplot( ___ ,Name,Value) returns the handles to the line objects and the
axes using optional name-value pair arguments.
