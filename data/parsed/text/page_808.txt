p.LowerBudget = 1;
p.UpperBudget = 1;
p.LowerBound = zeros(size(m));

plotFrontier(p);

Note Scenarios cannot be assigned directly using dot notation to a PortfolioMAD object. Scenarios
must always be set through either the PortfolioMAD object, the setScenarios function, or any of
the scenario simulation functions.
