Use setCosts to account for the impact of transaction costs when computing the return of the
portfolio.

% Transaction costs
cost = 3e-4; % 3 bps
w0 = zeros(p.NumAssets,1);
w0([1 2 3 4 16 18 19 20]) = 0.05;
w0(110) = 0.1;
w0(112) = 0.5;
p = setCosts(p,cost,cost,w0);

Use estimateFrontier to compute the efficient frontier and then use plotFrontier to plot the
efficient frontier of the portfolio problem.

% Plot efficient frontier
figure
pwgt = estimateFrontier(p);
plotFrontier(p,pwgt)

The plot shows that the minimum risk portfolio achieves a CVaR of 12.91% with an associated return
of 4.10%, while the maximum return portfolio has a CVaR of 30.93% and a return of 15.94%.
