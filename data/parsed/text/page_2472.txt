Input Arguments

Data — Data for low prices
matrix | table | timetable

Data for low prices, specified as a matrix, table, or timetable. Timetables and tables with M rows must
contain a variable named 'Low' (case insensitive).

Data Types: double | table | timetable

Name-Value Pair Arguments

Specify optional pairs of arguments as Name1=Value1,...,NameN=ValueN, where Name is the
argument name and Value is the corresponding value. Name-value arguments must appear after
other arguments, but the order of the pairs does not matter.

Before R2021a, use commas to separate each name and value, and enclose Name in quotes.

Example: values = llow(TMW_LOW,'NumPeriods',10)

NumPeriods — Moving window for the lowest low calculation
14 (default) | positive integer

Moving window for the lowest low calculation, specified as the comma-separated pair consisting of
'NumPeriods' and a scalar positive integer.

Data Types: double

Output Arguments

values — Lowest low series
matrix | table | timetable

Lowest low series, returned with the same number of rows (M) and the same type (matrix, table, or
timetable) as the input Data.
