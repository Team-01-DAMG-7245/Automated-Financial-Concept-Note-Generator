Overview

The sdeld class derives from the sdeddo class. The sdeld objects allow you to simulate correlated
paths of NVars state variables expressed in linear drift-rate form:

dXt = (A(t) + B(t)Xt)dt + D(t, Xt
α(t))V(t)dWt

sdeld objects provide a parametric alternative to the mean-reverting drift form, as discussed in
“Specify SDE from a Mean-Reverting Drift Model” on page 14-27. They also provide an alternative
interface to the sdeddo parent class, because you can create an object without first having to create
its drift and diffusion-rate components.

Specify SDELD Model

Create the same model as in “Specify Base Stochastic Differential Equation (SDE) Model” on page 14-

obj = sdeld(0, 0.1, 1, 0.3) % (A, B, Alpha, Sigma)
