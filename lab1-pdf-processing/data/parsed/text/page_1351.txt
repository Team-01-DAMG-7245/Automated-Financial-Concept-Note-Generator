Indicator to plot only active weights by category, specified as ActiveOnly and a logical with a value
of true (plot active weights only) or false (plot portfolio, benchmark, and active weights).

Data Types: logical

Output Arguments

h — Figure handle
handle object

Figure handle for weights by category chart, returned as handle object. You can use the figure handle
to access and change the properties of the chart.
