bndyield

Yield to maturity for fixed-income security

Syntax

Yield = bndyield(Price,CouponRate,Settle,Maturity)
Yield = bndyield( ___ ,Name,Value)

Description

Yield = bndyield(Price,CouponRate,Settle,Maturity) given NUMBONDS bonds with SIA
date parameters and clean prices (excludes accrued interest), returns the bond equivalent yields to
maturity.

Yield = bndyield( ___ ,Name,Value) adds optional name-value arguments.

Examples
