Notice that the inverse variance strategy begins all in cash and remains in that state for about 2.5
months. This is because the backtestStrategy object does not have a specified set of initial
weights, which you specify using the InitialPortfolioValue name-value pair argument. The
inverse variance strategy requires 30 days of trailing asset price history before rebalancing. You can
use the printRebalanceTable helper function, defined in the Local Functions on page 15-287
section, to display the rebalance schedule.

printRebalanceTable(ivStrategy,pricesTT,minLookback);
