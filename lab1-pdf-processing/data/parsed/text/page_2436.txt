AllRates — All the internal rates of return
serial date number | matrix

All the internal rates of return associated with CashFlow, returned as a matrix with the same
number of columns as CashFlow and one less row. Also, column j in AllRates contains all the rates
of return associated to column j in CashFlow (including complex-valued rates).

More About

Internal Rate of Return

Internal rate of return is a financial metric used to evaluate the profitability of an investment. It
represents the discount rate at which the net present value (NPV) of all cash flows (both incoming
and outgoing) from an investment equals zero.
