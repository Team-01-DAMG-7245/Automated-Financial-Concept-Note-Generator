depsoyd

Sum of years' digits depreciation

Syntax

Sum = depsoyd(Cost,Salvage,Life)

Description

Sum = depsoyd(Cost,Salvage,Life) computes the depreciation for an asset using the sum of
years' digits method.

Examples

Compute the Depreciation for an Asset Using the Sum of Years' Digits Method

This example shows how to calculate the depreciation for an asset using the sum of years' digits
method where the asset is $13,000 with a life of 10 years. The salvage value of the asset is $1000.

Sum = depsoyd(13000, 1000, 10)'
