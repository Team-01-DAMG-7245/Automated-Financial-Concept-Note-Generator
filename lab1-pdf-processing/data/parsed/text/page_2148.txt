depfixdb

Fixed declining-balance depreciation schedule

Syntax

Depreciation = depfixdb(Cost,Salvage,Life,Period)
Depreciation = depfixdb( ___ ,Month)

Description

Depreciation = depfixdb(Cost,Salvage,Life,Period) computes the fixed declining-
balance depreciation for each Period.

Depreciation = depfixdb( ___ ,Month) adds an optional argument.

Examples

Compute the Fixed Declining-Balance Depreciation

This example shows how to calculate the depreciation for the first five years for a car is purchased for
$11,000, with a salvage value of $1500, and a lifetime of eight years.

Depreciation = depfixdb(11000, 1500, 8, 5)
